# Branding (`/admin/branding`)

Personalize a **identidade visual** do portal Lotus para a agência ou white-label.

## O que pode ser alterado

| Item | Efeito |
| ---- | ------ |
| **Nome da marca** | Títulos de página, menu, e-mails |
| **Cores primárias** | Botões, destaques, gráficos |
| **Logo** | Cabeçalho e favicon (conforme implementação) |
| **Tom visual** | Variáveis CSS do design system Lotus |

## Passo a passo

1. Abra **Plataforma → Branding**.
2. Visualize o **preview** ao lado (se disponível).
3. Altere o **nome exibido** — ex.: “Agência XYZ Insights”.
4. Ajuste **cor primária** com o seletor ou hex.
5. Faça upload do **logo** (PNG/SVG, fundo transparente recomendado).
6. Clique **Salvar**.
7. Abra uma aba anônima e confira login + dashboard.

## Boas práticas

- Use contraste suficiente (acessibilidade WCAG).
- Logo horizontal funciona melhor no menu colapsado.
- Teste em dark mode se habilitado.
- Não altere branding em horário de apresentação ao cliente sem aviso.

## O que branding NÃO altera

- Dados de integração
- Permissões de usuário
- Estrutura de menus e rotas
