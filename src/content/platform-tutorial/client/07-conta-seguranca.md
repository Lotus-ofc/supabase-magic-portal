# Conta e segurança

Cuidar do acesso protege os dados da sua marca e evita atrasos em aprovações.

## Login

1. Use o **e-mail corporativo** cadastrado pela agência.
2. Senha forte: mínimo 10 caracteres, mistura de letras, números e símbolos.
3. Não compartilhe o mesmo usuário com 5 pessoas — peça contas individuais à agência.

## Trocar senha

1. Saia da conta (**Sair** no menu).
2. Na tela de login, use **Esqueci minha senha** (se habilitado).
3. Ou peça reset à agência pelo canal oficial.

## Sair com segurança

Sempre clique **Sair** em computadores compartilhados (recepção, coworking).

## O que cada pessoa deve ver

Sua conta mostra **apenas as marcas** vinculadas a você. Se enxergar dados de outra empresa, avise a agência imediatamente — é incidente de acesso.

## Dados e privacidade

- Métricas vêm de APIs oficiais (Google, Meta) com permissão da sua conta de anúncios.
- A plataforma **não publica** nada nas suas redes — só lê performance e gerencia aprovação de conteúdo.
- Conteúdos em **Aprovações** ficam armazenados para histórico da campanha.

## Quando pedir ajuda à agência

| Situação | O que informar |
| -------- | -------------- |
| Dashboard zerado | Marca, período, print da tela |
| Não recebeu convite | E-mail correto desejado |
| Erro ao aprovar post | Nome do card + print |
| Número “estranho” | Data, plataforma, KPI em dúvida |

## Atalhos úteis (resumo)

| Ação | Como |
| ---- | ---- |
| Busca | `Ctrl+K` (Windows) ou `⌘K` (Mac) |
| Tutorial | Menu → **Tutorial** |
| Aprovações urgentes | Menu → **Aprovações** → coluna aguardando |
| Visão rápida | Menu → **Visão geral** |
