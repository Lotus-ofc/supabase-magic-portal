# Aprovações

Área para **revisar o conteúdo** que a agência preparou antes de publicar. Você garante que texto, imagem e data estão corretos.

## Onde acessar

- `/aprovacoes` — modo padrão do seu usuário
- `/cliente/{sua-marca}/aprovacoes` — quando a agência envia link direto da marca

## Abas internas

| Aba | Uso para você |
| --- | ------------- |
| **Kanban** | Ver status de cada peça (produção → aguardando você → publicado) |
| **Calendário** | Ver **quando** cada post sai |
| **Pilares** | Temas editoriais (educação, produto, institucional…) |
| **Stories** | Grade de stories da semana |
| **Biblioteca** | Conteúdos já publicados / arquivo |

Comece pelo **Kanban** ou **Calendário**.

## Status que importam para você

| Status | Sua ação |
| ------ | -------- |
| aguardando_aprovacao | **Revise agora** |
| aprovado | Agência vai publicar na data |
| publicado | Já foi ao ar |
| producao / edicao | Ainda não chegou na sua fila — aguarde |

## Passo a passo: aprovar um post

1. Abra **Aprovações**.
2. No Kanban, vá à coluna **Aguardando aprovação** (ou equivalente).
3. **Clique no card** do post.
4. No painel lateral, confira:
   - **Imagem ou vídeo** — clique para ampliar
   - **Legenda** — ortografia, links, hashtags
   - **Data de publicação** — ok para sua agenda?
5. Se estiver tudo certo, clique **Aprovar**.
6. Se precisar mudança, clique **Reprovar** ou **Solicitar ajuste** e escreva o motivo no comentário.
7. A agência recebe o feedback na timeline do card.

## Passo a passo: comentar sem reprovar

1. Abra o card.
2. Campo **Comentário** — ex.: “Trocar CTA para WhatsApp”.
3. Envie — o card pode continuar em aguardando até você aprovar a versão corrigida.

## Calendário — visão mensal

1. Aba **Calendário**.
2. Navegue pelo mês.
3. Clique no dia — vê posts agendados.
4. Ideal para planejar campanhas e feriados.

## Prazos

- Aprove com **pelo menos 24–48h** de antecedência da data publicação.
- Reprovação em cima da hora pode atrasar a campanha.

## O que você não faz aqui

- Não sobe arquivo final de produção (agência faz).
- Não altera status manualmente arrastando colunas (modo admin) — use os botões Aprovar/Reprovar.
