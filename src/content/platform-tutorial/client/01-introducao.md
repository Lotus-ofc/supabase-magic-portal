# Introdução ao portal do cliente

Bem-vindo ao tutorial para **clientes** da plataforma Lotus. Aqui você aprende, passo a passo, como usar cada área disponível para a sua marca.

## Para quem é este guia

- Donos e gestores de marca
- Marketing e social media do cliente
- Qualquer usuário com papel **cliente** (não admin da agência)

Se você é da **equipe interna da agência**, use o tutorial em `/admin/tutorial`.

## Como fazer login

1. Acesse o link enviado pela agência.
2. Entre com **e-mail e senha** cadastrados.
3. Se o acesso estiver bloqueado, contate a agência — pode ser pendência contratual ou usuário não vinculado à sua marca.

## O que você vê no menu

| Aba | Rota | Função |
| --- | ---- | ------ |
| **Visão geral** | `/dashboard` | Resumo de investimento, conversões e performance |
| **Plano Estratégico** | `/plano-estrategico` | Objetivos e estratégias acordados com a agência |
| **Aprovações** | `/aprovacoes` ou `/cliente/{sua-marca}/aprovacoes` | Revisar e aprovar conteúdos |
| **Tutorial** | `/tutorial` | Este guia |

Se você tem mais de uma marca, a agência configura o acesso — você só vê os dados autorizados.

## Acesso interno (só se aparecer)

Alguns usuários admin da agência veem **Painel admin** no menu — ignore se você é cliente puro.

## Elementos da interface

### Período

Nos dashboards, escolha **7, 30 ou 90 dias**. As setas verde/vermelha comparam com o período anterior de mesmo tamanho.

### Tooltips ℹ

Passe o mouse para entender cada sigla (CTR, CPA, CPC, etc.).

### Busca (`Ctrl+K`)

Abre atalhos rápidos — rotas e glossário de métricas.

### Notificações

Ícone de sino — avisos da sessão (ex.: conteúdo aguardando aprovação).

## Ordem de leitura sugerida

1. **Visão geral** — entender seus números
2. **Painel da sua marca** — detalhe por plataforma
3. **Aprovações** — como aprovar posts
4. **Plano Estratégico** — alinhamento de longo prazo
